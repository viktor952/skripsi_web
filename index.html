Tentang Sistem

Sistem Informasi Geografis (SIG) Clusterisasi Vaksinasi COVID-19 di Kota Tobelo merupakan sistem berbasis web yang dikembangkan untuk memetakan dan menganalisis distribusi vaksinasi COVID-19 di wilayah Kota Tobelo, Maluku Utara.

Sistem ini menggunakan teknologi Geographic Information System (GIS) untuk memberikan visualisasi yang komprehensif tentang sebaran vaksinasi COVID-19, mengidentifikasi area yang terdampak dan tidak terdampak, serta melakukan analisis cluster untuk menentukan pola distribusi vaksinasi.

Fitur Utama

Dashboard Interaktif

Menampilkan statistik real-time tentang status vaksinasi dan data demografis

Peta Interaktif

Visualisasi geografis sebaran vaksinasi dengan berbagai layer informasi

Analisis Cluster

Pengelompokan area berdasarkan tingkat vaksinasi dan karakteristik demografis

Analisis Data

Grafik dan statistik untuk memahami tren vaksinasi

Teknologi yang Digunakan

Sistem ini dikembangkan menggunakan HTML5, CSS3, JavaScript, dan library Leaflet untuk pemetaan interaktif. Backend menggunakan PHP untuk pengolahan data dan API untuk komunikasi dengan database.
